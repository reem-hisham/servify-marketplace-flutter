# Servify Marketplace App 🚀

A full-service marketplace mobile application built with Flutter.

## 📌 Overview
Servify is a platform that connects users with nearby service providers (cleaning, maintenance, etc.).

## 🛠 Tech Stack
- Flutter
- Firebase (planned)
- Provider / Cubit
- REST APIs
